# Autor-2014-Replication
STATA replication of Autor (2014) regarding earnings inequality using CPS data.
Packages used: reghdfe, ftools, outreg2
Data Source: IPUMS CPS (1970–2015)
This sample replicates the core findings of Autor (2014) regarding the rising college premium. Using IPUMS CPS data (1970–2015), the syntax demonstrates data cleaning, the application of survey weights, and the use of high-dimensional fixed effects (reghdfe) to estimate wage gaps across educational cohorts.
